## 🛠 3. 使用方式 (快速上手)

請按照以下步驟完成資料路徑配置，即可成功載入數據：

### 📍 步驟一：準備資料夾
將下載的 `Sales_csv.zip` 檔案**解壓縮**至您的電腦中。
> 例如路徑為：`C:\Users\Documents\Project\Sales_csv`

### 📍 步驟二：啟動範本
雙擊開啟專案中的 **「我的作品.pbit」** 檔案。

### 📍 步驟三：輸入資料路徑
開啟檔案後，Power BI 會自動跳出 **「資料來源提示」** 視窗（如下圖意示）：

1.  在輸入框中，輸入您**解壓縮後的 Sales_csv 資料夾路徑**。
2.  點擊 **「載入 (Load)」**。

> [!IMPORTANT]
> **路徑輸入範例：** > `D:\MyData\Sales_csv`  
> *請確保路徑正確，Power Query 將會自動讀取該資料夾下的所有 CSV 檔案。*

## 📈 報表功能亮點
* **營收概覽**：即時掌握年度與月度銷售達成率。
* **產品分析**：找出 Top 5 熱銷產品與毛利貢獻。
* **地理分佈**：視覺化呈現各區域的銷售熱點。
